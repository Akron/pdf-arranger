# pdf-arranger

Web Component for PDF arrangements

# Usage

```html

<html>
  <head>
    <script type="module">
      import 'pdf-arranger';
    </script>
  </head>
  <body>
    <pdf-arranger url="example.pdf"></pdf-arranger>
  </body>
</html>
```



# Planned Features

- Select all / Unselect all
- Copy instead of moving
- Move/Copy without dragging and dropping
- Add files per drag and drop
